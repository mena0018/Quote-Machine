### Installation du projet

```bash
  git clone https://iut-info.univ-reims.fr/gitlab/mena0018/quote-machine.git
  cd quote-machine/
  make init
```

### Configuration de la base de données

```bash
  make database-init
```

### Lancement de l'application

```bash
  symfony server:start
  http://127.0.0.1:8000/   
```
